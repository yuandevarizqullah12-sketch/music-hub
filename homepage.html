Install Instructions

Android PWA Installation Guide

  1. Open Music Hub Web App in Chrome browser
  2. Tap the menu (three dots) in the top right corner
  3. Select "Add to Home screen" or "Install app"
  4. Follow the prompts to install the PWA
  5. Once installed, open the app from your home screen
  6. Enable background audio permissions when prompted

Note: Background play requires Android 8.0+ and Chrome 72+
