# Music Hub - YouTube Music Player

A modern, responsive YouTube Music player with Western music focus, built with Vercel Functions for secure API key handling.

## Features

- **Trending Western Music**: Discover popular Western songs
- **Advanced Search**: Search for songs, artists, or albums
- **Personal Library**: Save liked songs and manage playlists
- **Modern UI/UX**: Responsive design with smooth animations
- **Secure API Handling**: YouTube API keys protected via Vercel Functions
- **Mobile-First Design**: Works perfectly on all devices

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Vercel Serverless Functions (Node.js)
- **API**: YouTube Data API v3, YouTube IFrame Player API
- **Styling**: Custom CSS with CSS Variables, Animations
- **Storage**: LocalStorage for user preferences
